# AI ITSM Copilot

A ServiceNow-style IT service management copilot prototype for incident triage and support recommendation.

## What it does

This prototype takes a new support ticket and generates:

- recommended category
- recommended urgency
- recommended assignment group
- similar past incident
- relevant knowledge-base articles
- suggested resolution
- user-facing response draft
- internal work note for support agents
